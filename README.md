# CDK Lambda APIGW DDB

This is a demo showing how to deploy an API Gateway (HTTP API) with multiple TypeScript Lambdas, backed by separate Dynamo DB tables


## Prerequisites

- Docker
- Node 18+
- awscli


## Installation

1. Clone this repository

2. Install NPM dependencies

`npm install`


## Deploy

1. Deploy API Stack

`cdk deploy ApiStack`
